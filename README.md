# StyleStore

StyleStore is a simple web application that showcases a fashion and makeup store. It includes a homepage with a hero section and a product listing page. The project is built using HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: The layout adjusts to different screen sizes.
- **Hero Section**: A visually appealing introduction to the store.
- **Product Listing**: Displays products in a grid layout with images, names, and prices.
- **Navigation**: Links to switch between the homepage and the product listing.

## File Structure

- `StyleStore.html`: The main HTML file containing the structure and content of the web application.
- Inline CSS: Styles are defined within the `<style>` tag in the `<head>` section.
- Inline JavaScript: Functions to toggle between the homepage and product listing are included in the `<script>` tag.

## How to Use

1. Open the `StyleStore.html` file in any modern web browser.
2. Use the navigation links in the header to switch between the "Home" and "Shop" sections.
3. Click the "Shop Now" button in the hero section to view the products.

## Technologies Used

- **HTML**: For the structure of the web page.
- **CSS**: For styling the layout and design.
- **JavaScript**: For interactivity, such as toggling between sections.
